Streamer Tablet Helper - PC Server  v4.0
=========================================

QUICK START - Two options:

1. THE EASY WAY (Run the EXE)
   - Go to the 'dist' folder
   - Double-click StreamerTabletHelper.exe
   - That's it! No Python needed.

2. THE DEV WAY (Run the script)
   - Run 'install_deps.bat' to install requirements
   - Run 'streamer_helper_server.py' with Python 3

-----------------------------------------
FEATURES:
- Keyboard shortcuts: simulate any key combo
- Sounds: play any .wav file
- OBS Control: scenes, streaming, recording, mute
- Twitch Markers: create stream markers instantly
- Auto-Start: option to start with Windows
- System Tray: runs in the background

-----------------------------------------
HOW TO USE WITH THE TABLET:
1. Run the server on your PC
2. It will show your IP address (e.g., 192.168.1.50)
3. Open the app on your tablet/phone
4. Enter that IP address and click 'Connect'
5. Your buttons will appear!

-----------------------------------------
TWITCH SETUP:
- Open 'Settings' in the PC app
- Enter your Twitch Client ID (get one at dev.twitch.tv)
- Click 'Log in with Twitch'
- A browser will open - follow the steps
- Now your Twitch marker buttons will work!
