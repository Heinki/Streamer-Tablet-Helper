  Streamer Tablet Helper — PC Server  v4.0
═══════════════════════════════════════════════════════

QUICK START — Two options:

OPTION A: Run from Python (if you have Python installed)
────────────────────────────────────────────────────────
1. Install dependencies (one time only):
   Double-click:  install_deps.bat

2. Run the server:
   Double-click:  streamer_helper_server.py
   (or right-click → Open With → Python)


OPTION B: Build a standalone .EXE (no Python needed after)
────────────────────────────────────────────────────────────
1. Install Python from https://python.org  (one time only)
2. Double-click:  build_exe.bat
3. Your EXE appears in the dist/ folder
4. Copy StreamerTabletHelper.exe anywhere you like
5. Double-click the EXE — that's it, no Python needed


WHAT THE APP DOES
─────────────────
• Shows your PC's IP address — type that into the tablet/phone app
• Receives button presses from your devices over local Wi-Fi
• Executes keyboard shortcuts, sounds, OBS commands, Twitch markers
• Settings panel built in — no terminal ever needed
• "Start with Windows" toggle in Settings


OBS INTEGRATION
───────────────
• Open OBS → Tools → WebSocket Server Settings → Enable
• Set a password
• Enter that password in the app's Settings tab


TWITCH MARKERS
──────────────
• Go to dev.twitch.tv/console → Register Application
  Name: anything · Type: Public · Redirect: http://localhost
• Copy the Client ID into the Settings tab
• Click "Log in with Twitch" — browser opens automatically
• Approve once, done forever (token auto-refreshes)
